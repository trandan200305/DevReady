# Lộ trình phát triển (Roadmap)

Dự án DevReady được chia thành 3 giai đoạn lớn, phát triển tuần tự từ phân tích tĩnh đến can thiệp động vào hệ thống.

## Giai đoạn 1: Lõi phân tích (Core Analysis) - Hiện tại
- **Phiên bản 1.0 - Hardware Scanner**: Xây dựng UI cơ bản, quét thành công toàn bộ CPU, RAM, GPU, Disk.
- **Phiên bản 1.1 - Software Scanner**: Quét các ứng dụng IT phổ biến (IDE, Database, Tools).
- **Phiên bản 1.2 - Runtime Scanner**: Quét hệ sinh thái .NET, Java, Python, C++.
- **Phiên bản 1.3 - Compatibility Analysis**: Ra mắt tính năng chấm điểm tương thích (0-100%).
- **Phiên bản 1.4 - AI Advisor**: Tích hợp thành công API OpenAI/Gemini, hiển thị lời khuyên giao diện chat/card.
- **Phiên bản 1.5 - Report Generator**: Hỗ trợ xuất PDF và Excel.

## Giai đoạn 2: Phân tích chuyên sâu (Deep Analysis)
- **Phiên bản 2.0 - Project Analyzer**: Cho phép thả (Drag & Drop) thư mục code vào app để tự động biết máy có chạy được project đó không.
- **Phiên bản 2.1 - Driver Analyzer**: Cảnh báo Driver lỗi thời hoặc không tương thích.
- **Phiên bản 2.2 - Performance Prediction**: Dự đoán FPS game, dự đoán tốc độ Build code dựa vào mô hình dữ liệu.

## Giai đoạn 3: Trợ lý chủ động (Active Assistant)
- **Phiên bản 3.0 - Auto Fix**: 1 Click để tự động tải về và cài đặt ngầm các Runtime/SDK/Driver còn thiếu (sử dụng winget hoặc các tool CLI nội bộ).
- **Phiên bản 3.1 - Cloud Sync**: Đăng nhập tài khoản, lưu trữ lịch sử cấu hình máy trên đám mây để so sánh.

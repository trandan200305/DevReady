# Thiết kế Giao diện (UI Design)

PC Insight hướng đến giao diện hiện đại, sạch sẽ và công nghệ (tech-oriented), lấy cảm hứng từ các Dashboard chuyên nghiệp.

## 1. Hướng dẫn thiết kế (Design Guidelines)
- **Theme chủ đạo**: Dark Mode (chế độ tối) để giảm mỏi mắt cho lập trình viên/game thủ.
- **Màu sắc (Color Palette)**:
  - Nền (Background): `#1E1E2F` hoặc `#181824`.
  - Màu nhấn (Accent Color): Xanh lam (Blue `#3B82F6`) hoặc Xanh ngọc (Teal `#14B8A6`).
  - Lỗi/Cảnh báo: Đỏ (`#EF4444`) / Vàng (`#F59E0B`).
- **Typography (Font chữ)**:
  - Sử dụng `Segoe UI Variable` (trên Win 11) hoặc các font như `Inter`, `Roboto` cho tính hiện đại.

## 2. Cấu trúc Màn hình (Screens)
1. **Trang chủ (Dashboard)**: 
   - Tổng quan sức mạnh hệ thống.
   - Nút "QUÉT HỆ THỐNG" nổi bật ở trung tâm.
2. **Trang Chi tiết Phần cứng (Hardware Details)**: 
   - Biểu đồ phân bổ RAM.
   - Thông số chi tiết CPU, GPU.
3. **Trang Tương thích (Compatibility Checker)**:
   - Thanh tìm kiếm để gõ tên ứng dụng/game.
   - Hiển thị điểm số tương thích (Thanh progress bar vòng cung - Circular Progress).
4. **Trang Tư vấn AI (AI Advisor)**:
   - Hiển thị giống một đoạn hội thoại hoặc các thẻ (Cards) lời khuyên, có đánh dấu ưu tiên màu Đỏ/Vàng/Xanh.

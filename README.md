# PC Insight

## 📖 Project Overview

PC Insight is a Windows desktop application developed with C# and WPF that analyzes a computer's hardware, software, and development environment.

The application scans the user's computer, evaluates whether it meets the requirements for running applications, games, or development tools, detects missing runtimes and software dependencies, and provides AI-powered recommendations.

Unlike traditional system information tools, PC Insight not only displays hardware information but also explains what the information means and how users can improve their systems.

## 🚀 Main Features

1. **Hardware Scanner**: Scans CPU, RAM, GPU, Storage, Motherboard, and Windows Version.
2. **Software Scanner**: Detects installed software such as Visual Studio, SQL Server, Git, Docker, Python, Java, and NodeJS.
3. **Runtime Scanner**: Detects .NET Runtime, .NET SDK, Java Runtime, Python, Visual C++, and DirectX.
4. **Compatibility Checker**: Evaluates whether the PC can run Desktop Applications, Development Tools, and Games. Generates a Compatibility Score (0% - 100%).
5. **Development Environment Analyzer**: Analyzes whether the PC is ready for development (e.g., checks for Visual Studio, Git, SQL Server, Docker, NodeJS).
6. **AI Advisor**: Analyzes scan results and generates recommendations (e.g., hardware upgrades, missing runtimes).
7. **Report Generator**: Generates reports in PDF and Excel formats.

---

## 🧠 AI Functions

AI is NOT responsible for scanning the PC. The application performs the scan. AI analyzes the scan results and provides intelligent recommendations in natural language.
